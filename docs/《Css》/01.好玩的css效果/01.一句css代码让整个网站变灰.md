## 用一句代码给网站加上`灰色模式`

#### 在任意网站中，打开浏览器开发者工具(F12)，在Console控制台输入如下代码并回车：
```js
document.documentElement.style.filter='grayscale(100%)'
```
#### 当前打开的网站变成了灰色
